About Strava Segment Wind Analysis
This is the deprecated Strava Wind Analysis website. The purpose of this website is to compare and benchmark different segment efforts and determine if wind was a major factor in the distribution of KOMs. It accomplishes this by using vector manipulation and reading data from external APIs.

The new website features an improved user interface, and a better algorithm for determining wind's impact on performance. Check it out here.
